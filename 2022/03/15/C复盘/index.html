for循环

for循环，我们使用的for循环中变量要先定义，

int i,sum=0;

for(i=1;i<=100;i++){

sum=sum+i;

}

执行过程：因为我是1+2+3+…+100;

我先要块地，来放最后的值。

所以我肯定要定义个变量来接收。

然后i通过for循环，i的变化是1,2,3,4…100

因为我们一次只蹦跶一个。

如果我们需要一次蹦跶2个，比如求奇数和或者偶数和的时候，我们只需要修改第三个位置

将原来的i++改成i=i+2;或者i+=2;这两个表达的意思是一样的，都是实现i蹦跶两个。

我们还可以在for循环里添加一些判断条件，比如我们以前遇到过的整除问题

我们还可以在for循环里添加小for循环，比如打印星星问题，输出几排几列的星星。

函数：

注意函数的格式

首先我们要声明

Int max（int a,int b）;

这个作用告诉计算机我有这么一个函数你以后会用到，不写的话，计算机不认识他，你是谁啊？就会报错。

然后我们自己定义函数

Int max(int a,int b){

Int sum;

函数体内容；

Return sum;

}

函数体内容，具体就是对你穿过来的东西进行操作，也就是a和b，对你们操作完，ok,我扔给你个东西，return 给你,

我们需要使用函数的功能时，由于函数他会扔个你个东西，所以你在使用函数以前，需要先向计算机要块地，来存放你函数给我的东西。调用的方式是：

Int a;

a=max(b,c);

这里的b,c是真实的值，扔给函数，进行操作。
